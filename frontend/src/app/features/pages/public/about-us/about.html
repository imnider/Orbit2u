Orbit2u

Plataforma de contenido multimedia y comunidades colaborativas

El proyecto

Orbit2u permite la publicación y visualización de contenido multimedia mediante una API REST construida en ASP.NET Core, integrando autenticación JWT y persistencia en SQL Server.

Incorpora un sistema de comunidades donde el canal moderador puede compartir contenido en espacios colaborativos — útil para entornos académicos, cursos, grupos de estudio o comunidades de intereses.

La plataforma incluye un enfoque de personalización visual con una interfaz inspirada en estética espacial y múltiples temas configurables para mejorar la experiencia del usuario.
